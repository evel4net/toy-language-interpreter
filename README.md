# Toy Language Interpreter
*Note: Project in progress*

A toy language interpreter implemented in Java for the **Advanced Programming Methods** course at *Babe»ô-Bolyai University* during the first semester of second year (2025 - 2026).

## Examples of programs
```
int a;
a=2+3*5;
int b;
b=a-4/2+7;
Print(b)
```
```
bool a;
a=false;
int v;
If a Then v=2 Else v=3;
Print(v)
```
